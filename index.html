HOME PAGE

This is wikipedia api home page. There are several methods you can use to get different data in json. This api is made in Python using wikipedia module and Flask . The api have several endpoint and routes, displayed below

First Endpoint is "content"

This endpoint when used returns content and suggestions of the query you entered.

Use it as: /content/?q="query"
For example: /content/?q=india

Second Endpoint is "page"

This endpoint when used returns summary, images and suggestions of the query you entered.

Use it as: /page/?q="query"
For example: /page/?q=india

Third Endpoint is "search"

This endpoint when used returns search suggestions of the query you entered.

Use it as: /search/?q="query"
For example: /search/?q=india
